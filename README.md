#  Project Overview


An interactive **Power BI dashboard** built to analyze Apple's global sales, product performance, and warranty operations. The dashboard helps users monitor revenue, product performance, warranty claims, and regional quality metrics through interactive visualizations.


#  Purpose

The Apple Global Sales & Warranty Analytics Dashboard is a **3-page interactive Power BI report** designed to provide a complete overview of Apple's global business performance. It helps analyze sales trends, product insights, and warranty operations, enabling users to identify high-performing products, monitor warranty claims, and evaluate business performance across different countries and product categories.


# Tech Stack

The dashboard was built using the following tools and technologies:

* 📊 **Power BI Desktop** – Used to build interactive reports and dashboards.
* 🔄 **Power Query** – Used for data cleaning and transformation.
* 🧠 **DAX (Data Analysis Expressions)** – Used for KPIs, calculated measures, and business metrics.
* 🔗 **Data Modeling** – Built relationships between Sales, Warranty, Product, and Date tables.
* 📅 **Calendar Table** – Used for time intelligence calculations like Year-over-Year Growth.
* 📁 **File Format** – `.pbix` for development and `.png` for dashboard previews.


#  Data Source

The dataset contains information about:

* Product Sales
* Revenue
* Units Sold
* Product Categories
* Warranty Claims
* Repair Status
* Countries
* Dates

The data represents Apple's global sales and warranty operations across multiple countries and product categories.


## • Business Problem

Companies selling products globally need to monitor both **sales performance** and **product quality**. With large volumes of sales and warranty claims, it becomes difficult to identify top-performing products, monitor claim trends, and track operational performance using raw data alone.

## • Goal of the Dashboard

The dashboard was created to:

* Monitor overall sales performance.
* Analyze product revenue and category performance.
* Track warranty claims and repair status.
* Identify high-risk products and regions.
* Support business decisions using interactive reports.
